# Big O notation
- It's used to measure the performance of an algorithm
- Syntax : `O(<something>)`
- Types :
	1. `O(1)` : when execution doesn't depend on a var (fastest) called `constant time`
	2. `O(n)` : when execution depend on a var, can be seen in loops (linear)
	3. `O(n^2)` : seen in nested loops (exponential)
